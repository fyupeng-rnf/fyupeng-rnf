# RNF 是什么？

## RNF 简介

### 概览

RNF（全称：RPC-Netty-Framework）是一款动态远程方法调用的 `RPC` 框架。

### 解决方案

• 异步非阻塞通信的解决方案；<br/>
• 动态代理远程调用解决方案；<br/>
• IO 密集型和CPU 密集型场景的应用；<br/>
• 分布式的超时重试机制解决方案；<br/>
• 雪花算法的 id 生成器解决方案;<br/>
• SPI 机制解决方案；<br/>
• 高可用性解决方案；<br/>
• 负载均衡策略解决方案；<br/>
• 序列化方式解决方案； <br/>
• 安全校验机制解决方案；<br/>
• TCP 长连接、低延时解决方案；<br/>
• 服务池中服务增强解决方案； <br/>
• 自定义应用层通信协议解决方案;<br/>
